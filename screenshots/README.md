# Screenshots for GitHub README

To make your repository look highly professional, run your project locally and take screenshots of the following pages. Save the images in this `screenshots/` directory with the exact filenames listed below so they link correctly in the main `README.md` file:

1. **`banner.png`** - A wide banner image (optional, can be a nice library background or logo).
2. **`login.png`** - The landing/login page showing both Admin and Student login options.
3. **`admin_dashboard.png`** - The dashboard view after logging in as an administrator.
4. **`book_management.png`** - The page where books can be added, updated, or deleted.
5. **`student_portal.png`** - The user portal view for students where they can search books and check their profile.

Once you have added these files to this folder, they will automatically show up when you push the project to GitHub!
